# Portfolio

Portfolio starter built with Vite + React + TypeScript, based on a dark single-page layout.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Sections

- Hero intro
- About
- Values
- Ambitions
- Contact (Email + GitHub)

## Notes

- Top navigation links (`About`, `Values`, `Ambitions`, `Contact`) scroll smoothly to each section.
- Main design/styling is in `src/styles/globals.css`.
